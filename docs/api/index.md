# API Reference

Complete API documentation for Pantanal Editor components and types.

## Components

### [PantanalEditor](/api/pantanal-editor)

The main editor component that provides rich text editing capabilities.

### [EditorTool](/api/editor-tool)

Component for creating custom toolbar tools with full control over behavior and appearance.

## Types

### [EditorCommand](/api/types)

Available editor commands for programmatic control.

### [ToolbarItem](/api/toolbar-item)

Interface for defining toolbar items and custom tools.

## Props

### [DeserializationProps](/api/DeserializationProps)

Custom deserialization function for processing HTML on load.

### [SerializationProps](/api/SerializationProps)

Custom serialization function for processing HTML on save.
