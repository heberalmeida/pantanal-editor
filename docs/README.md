# Pantanal Editor Documentation

This directory contains the documentation for Pantanal Editor, built with VitePress.

## Development

```bash
npm run docs:dev
```

## Build

```bash
npm run docs:build
```

## Preview

```bash
npm run docs:preview
```

## Structure

- `/guide` - Getting started and guides
- `/api` - API reference documentation
- `/examples` - Code examples and demos

## Contributing

When adding new documentation:

1. Follow the existing structure
2. Include code examples
3. Add links to related pages
4. Keep examples functional and tested
